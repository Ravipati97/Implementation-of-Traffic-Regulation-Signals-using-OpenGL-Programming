Implementation of Traffic Regulation Signals using OpenGL Programming

Instructions to run the code: 
1. Please include the following library files in Visual Studio the linker/additional dependencies/input section in the properties-
   freeglutd.lib
   glew32d.lib
   assimp-vc140.mt.lib
   SOIL.lib

2. Also include the following directories in the VC++ directories/include directories section in the properties as per your system paths.
   The following directories are as per my system requirements -
   C:\Program Files (x86)\glm
   C:\Program Files (x86)\Simple OpenGL Image Library\src

3. Build the code.

4. Run the code "Local Windows Debugger"
